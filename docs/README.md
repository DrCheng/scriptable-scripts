# LynkCo Scriptable Widgets


> 基于 `Scriptable` App 开发的一个桌面小组件脚本。使用iOS 小组件快速开发框架「小件件」作为开发框架而搭建。


主要从吉利 GKUI API中获取车辆相关信息并以桌面小组件的形式展示到iOS设备上。
